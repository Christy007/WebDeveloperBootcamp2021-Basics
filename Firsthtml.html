  3. Anchor tag

    < a> is anchor tag
    Anchor tags are defined with the href attribute. Attribues are special words inside a open tag which controls the elements behaviour
    href attribute specifies the link to which the anchor tag has to be redirected.

  4. Image


    < img> is used to add image to our webpage.
    Main attributes for img element are src to sepecify the location of the image. It can be local pictures or any online links
    Next attribute is alt which describes the image making it more accessible to all users using screen readers

  5. Span and div containers

    < div>: Content Division Element: basically a container to group elements together. It's a block level element(which takes up the whole line)
    < span>It's also a container, usually used to group elements ot texts in the same line.

  6. Thematic and Line breaks

    < hr> represents a thematic break between paragraph elements as implemented before and after the nav section at the begining of this webpage
    < br> is used as a line break in paragraph

  7. Superscript and Subscript

    < sup> Superscripts are usually rendered with a raised baseline using smaller text.
    < sub> Subscripts are typically rendered with a lowered baseline using smaller text
    Examples:

    1. x2 + y2
    2. C8H10N4O2

  8. Entity codes

    With entity code we can display symbols or specials characters in our html to build a strucuted html code
    Usually starts with ampersand and ends with a semicolon, example: &_amp_; = &
    Browser will intrpret this code and will render the correct code

  9. Time

    < time> represents a specific period in time. datetime attribute translates date into machine readable format
    Example: next interview preparation class is on < time datetime="2021-06-08">8th June< /time> which starts at < time datetime="18:00">18:00< /time>
    With the help of this element we can make our code more semantic.

  10. Figure

    This is again a html element that we can use to make our code more semantic. Using this element we can containerize image and its caption together

  11. Table

    It is structured set of data built using rows and columns
    < caption> displays the caption for the table
    < td> represents a cell of the table that contains data
    < tr> represents row of cells in a table
    < th> represents a cell as header. Text entered in in this cell are default Bold
    < thead> defines set of rows defining the head of the columns
    < tbody> defines set of rows that below to the body of the table
    < tfoot> defines a set of rows summarizing the columns of the table
    colspan this attribute indicates how many columns the cell extends
    rowspan this attribute indicates how many rows the cell extends

    Sample table structure
    Header1 Header2 with colspan = 2 Header3
    Cell1 in Row1 Cell2 in Row1 Cell3 in Row1 Cell4 in Row1
    Cell1 in Row2 Cell2 in Row2 Cell3 in Row2 Cell4 in Row2
    cell with rowspan=2 footer1 cell2 footer1 cell3 footer1 cell4
    footer2 cell2 footer2 cell3 footer2 cell4

  12. Form

    < form> represents a document section containing interactive controls for submitting information
    main attributes of form are:
    1. action : which specifies where the form data should be sent.
    2. method : specifies which HTTP method to be used.

    < input> element is used to create different form controls
    type attribute alters the input's behaviour and appearance
    search attribute value for type element represents a single-line text field for entering search strings. May include a delete icon in supporting browsers that can be used to clear the field
    checkbox attribute value for type element allows single values to be selected/deselected.
    radio attribute value for type element allows a single value to be selected out of multiple choices with the same name value.
    range attribute value for the type element displays as a range widget.
    placeholder attribute gives the input field a default text.
    < label> element is used to link the input/form control to a text/label. Value of for attribute in label tag should match with the value of id attribute in respective input tag
    < button> element declared inside a form is by default of submit type. To chnage the functionality of the button inside a form we use the attribute type. type="button" make a button inside a form behave just like a normal button.
    name attribute helps to link the data in the input fields with the query parameters of the server, when a form is submitted
    < select> element creates a dropdown and the values in the dropdown is defined by option tag
    < textarea> element represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback
